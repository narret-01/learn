git status - проверка состояния 

git add file.xxx - обозначаем новый файл - как имеющий отношение к проекту

git commit -m "blabla" - добавляем комментарий про изменение

git push - закидываем данные в облако ( закинуть новыею информацию )

git pull - обновляет локальный репозиторий новыми данными из облака ( скачать новую информацию )

git add . - добавить все неотслеживаемые файлы сразу

// все зеленые файлы готовы к коммиту - отслеживаются гитом
